# 🏠 House Price Prediction Using Machine Learning

## 1. 🎯 Objective
Predict house prices based on features like location, area, and amenities to understand real estate pricing trends.

## 2. 🛠 Tools & Technologies
- Python (Pandas, Scikit-learn, Matplotlib, Seaborn)  
- Excel  

## 3. 📊 Output / Result
- Model predictions for house prices.  
- Visualizations showing feature importance and predicted vs actual prices.  
- Performance metrics: RMSE, MAE, R² score.  

## 📁 Project Structure
/data → Raw & cleaned datasets  
/notebooks → Python scripts  
/images → Charts & visualizations
