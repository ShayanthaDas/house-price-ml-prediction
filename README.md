# House Price Prediction (Machine Learning Project)

## Overview

This project is about predicting house prices using machine learning based on different property features like size, location, number of rooms, and other factors.

The main idea was to understand what really affects house prices and build a simple model that can make reasonable predictions.

## What this project is about

* Predicting house prices using regression models
* Understanding key factors that influence property value
* Practicing end-to-end machine learning workflow
* Learning how to handle real-world structured data

## Tools used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

## Workflow

1. Data loading and inspection
2. Data cleaning and handling missing values
3. Exploratory data analysis
4. Feature selection and encoding
5. Model building
6. Model evaluation
7. Prediction

## Models used

* Linear Regression
* Ridge / Lasso Regression
* Random Forest Regressor

## Key insights

* Location has a strong impact on house prices
* Bigger houses generally have higher prices, but not always
* Certain features like quality and condition matter a lot
* Simple models already give decent baseline performance

## Results

The model was able to make fairly accurate price predictions after proper preprocessing and feature selection.
